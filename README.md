# todo-discord

![](https://img.shields.io/github/license/kissy24/todo-discord)

It is a bot that registers and notifies todo on discord.

## Usage

### Backend

```sh

rye run devserver

```

### Discord Bot

```sh

rye run python bot/bot.py

```

### Discord

```text

!add_todo Hello World!

```
